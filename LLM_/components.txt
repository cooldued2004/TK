в”Ӯ   в”Ӯ   в””в”Җв”Җ Unicode normalization
в”Ӯ   в”Ӯ       в”ңв”Җв”Җ Character decomposition
в”Ӯ   в”Ӯ       в””в”Җв”Җ Codepoint mapping
в”Ӯ   в””в”Җв”Җ Subword segmentation (BPE/WordPiece)
в”Ӯ       в”ңв”Җв”Җ Pair frequency counting
в”Ӯ       в”ңв”Җв”Җ Pair merging
в”Ӯ       в””в”Җв”Җ Token boundary updates
в”Ӯ   в””в”Җв”Җ Vocabulary lookup
в”Ӯ       в”ңв”Җв”Җ Hash table or trie search
в”Ӯ       в”Ӯ   в”ңв”Җв”Җ String comparison
в”Ӯ       в”Ӯ   в””в”Җв”Җ Index retrieval

в”Ӯ       в”ңв”Җв”Җ Array indexing
в”Ӯ       в””в”Җв”Җ Integer assignment

в”Ӯ       в””в”Җв”Җ comparison, addition, indexing, assignment

в”ңв”Җв”Җ 2. Embedding Layer
в”Ӯ   в”ңв”Җв”Җ Token embedding
в”Ӯ   в”Ӯ   в”ңв”Җв”Җ Token embedding matrix [Vocab_size Г— Hidden_dim]
в”Ӯ   в”Ӯ   в”ңв”Җв”Җ Array indexing вҶ’ vector
в”Ӯ   в”Ӯ   в””в”Җв”Җ Copying numbers
в”Ӯ   в””в”Җв”Җ Positional embedding
в”Ӯ       в”ңв”Җв”Җ Position embedding matrix [Max_length Г— Hidden_dim]
в”Ӯ       в”ңв”Җв”Җ Array lookup
в”Ӯ       в””в”Җв”Җ Vector addition (element-wise FP addition)
в”Ӯ   в””в”Җв”Җ Bottom-most operations
в”Ӯ       в””в”Җв”Җ memory fetch, addition, FP arithmetic
в”Ӯ
в”ңв”Җв”Җ 3. Transformer Decoder Stack (N layers)
в”Ӯ   в”ңв”Җв”Җ Decoder Layer
в”Ӯ   в”Ӯ   в”ңв”Җв”Җ A. Masked Multi-Head Self-Attention
в”Ӯ   в”Ӯ   в”Ӯ   в”ңв”Җв”Җ Linear Projections (Q, K, V)
в”Ӯ   в”Ӯ   в”Ӯ   в”Ӯ   в”ңв”Җв”Җ Weight matrix [Hidden_dim Г— Hidden_dim]
в”Ӯ   в”Ӯ   в”Ӯ   в”Ӯ   в”ңв”Җв”Җ Bias vector [Hidden_dim]
в”Ӯ   в”Ӯ   в”Ӯ   в”Ӯ   в””в”Җв”Җ Matrix multiplication
в”Ӯ   в”Ӯ   в”Ӯ   в”ңв”Җв”Җ Attention Score Computation
в”Ӯ   в”Ӯ   в”Ӯ   в”Ӯ   в”ңв”Җв”Җ Q В· KбөҖ [Seq_len Г— Seq_len] (dot products)
в”Ӯ   в”Ӯ   в”Ӯ   в”Ӯ   в”ңв”Җв”Җ Scale by 1/вҲҡd_k
в”Ӯ   в”Ӯ   в”Ӯ   в”Ӯ   в”ңв”Җв”Җ Masking (compare positions вҶ’ assign -вҲһ)
в”Ӯ   в”Ӯ   в”Ӯ   в”Ӯ   в””в”Җв”Җ Softmax (exp вҶ’ sum вҶ’ divide)
в”Ӯ   в”Ӯ   в”Ӯ   в”ңв”Җв”Җ Weighted Value Aggregation
в”Ӯ   в”Ӯ   в”Ӯ   в”Ӯ   в””в”Җв”Җ Attention weights Г— V (matrix mult)
в”Ӯ   в”Ӯ   в”Ӯ   в”ңв”Җв”Җ Multi-Head Concatenation
в”Ӯ   в”Ӯ   в”Ӯ   в””в”Җв”Җ Output Linear Projection (matrix mult + bias)
в”Ӯ   в”Ӯ   в”Ӯ   в””в”Җв”Җ Bottom-most: dot products, mult, add, div, exp, memory access
в”Ӯ   в”Ӯ   в”ңв”Җв”Җ B. Residual Connection
в”Ӯ   в”Ӯ   в”Ӯ   в””в”Җв”Җ Element-wise addition
в”Ӯ   в”Ӯ   в”ңв”Җв”Җ C. Layer Normalization
в”Ӯ   в”Ӯ   в”Ӯ   в”ңв”Җв”Җ Compute mean (sum / count)
в”Ӯ   в”Ӯ   в”Ӯ   в”ңв”Җв”Җ Compute variance (sum((x-mean)ВІ) / count)
в”Ӯ   в”Ӯ   в”Ӯ   в”ңв”Җв”Җ Normalize: (x-mean)/вҲҡ(variance+Оө)
в”Ӯ   в”Ӯ   в”Ӯ   в””в”Җв”Җ Scale/shift (Г—Оі + ОІ)
в”Ӯ   в”Ӯ   в”Ӯ   в””в”Җв”Җ Bottom-most: sub, add, mult, div, sqrt, exp
в”Ӯ   в”Ӯ   в”ңв”Җв”Җ D. Feed-Forward Network (FFN)
в”Ӯ   в”Ӯ   в”Ӯ   в”ңв”Җв”Җ Linear Layer 1
в”Ӯ   в”Ӯ   в”Ӯ   в”Ӯ   в”ңв”Җв”Җ Weight matrix + bias вҶ’ matrix mult
в”Ӯ   в”Ӯ   в”Ӯ   в”Ӯ   в””в”Җв”Җ GELU activation
в”Ӯ   в”Ӯ   в”Ӯ   в”Ӯ       в”ңв”Җв”Җ 0.5 Г— x Г— (1 + tanh(вҲҡ(2/ПҖ) Г— (x + 0.044715Г—xВі)))
в”Ӯ   в”Ӯ   в”Ӯ   в”Ӯ       в””в”Җв”Җ mult, add, exp, div, tanh
в”Ӯ   в”Ӯ   в”Ӯ   в””в”Җв”Җ Linear Layer 2 (matrix mult + bias)
в”Ӯ   в”Ӯ   в”Ӯ   в””в”Җв”Җ Bottom-most: dot product, add, mult, exp, tanh, div
в”Ӯ   в”Ӯ   в”ңв”Җв”Җ E. Second Residual Connection
в”Ӯ   в”Ӯ   в”Ӯ   в””в”Җв”Җ Element-wise addition
в”Ӯ   в”Ӯ   в””в”Җв”Җ F. Second Layer Normalization
в”Ӯ   в”Ӯ       в””в”Җв”Җ (Same as C)
в”Ӯ   в”ңв”Җв”Җ 4. Final Layer Normalization
в”Ӯ   в”Ӯ   в””в”Җв”Җ (Same as layer norm)
в”Ӯ   в””в”Җв”Җ 5. Output Projection Layer
в”Ӯ       в”ңв”Җв”Җ Linear projection вҶ’ logits
в”Ӯ       в”ңв”Җв”Җ Weight matrix [Hidden_dim Г— Vocab_size]
в”Ӯ       в”ңв”Җв”Җ Bias vector [Vocab_size]
в”Ӯ       в””в”Җв”Җ Matrix mult + element-wise add
в”Ӯ
в”ңв”Җв”Җ 6. Probability Layer (Softmax)
в”Ӯ   в”ңв”Җв”Җ Exponentiation
в”Ӯ   в”ңв”Җв”Җ Sum of exponentials
в”Ӯ   в””в”Җв”Җ Division вҶ’ normalize probabilities
в”Ӯ   в””в”Җв”Җ Bottom-most: exp, add, div
в”Ӯ
в””в”Җв”Җ 7. Decoding / Sampling
    в”ңв”Җв”Җ Greedy selection
    в”Ӯ   в”ңв”Җв”Җ Compare probabilities
    в”Ӯ   в””в”Җв”Җ Choose maximum
    в”ңв”Җв”Җ Top-k sampling
    в”Ӯ   в”ңв”Җв”Җ Sort probabilities
    в”Ӯ   в””в”Җв”Җ Choose from top-k
    в””в”Җв”Җ Nucleus sampling
        в”ңв”Җв”Җ Temperature scaling (logits Г— 1/T)
        в”ңв”Җв”Җ Compute cumulative probability
        в””в”Җв”Җ Select where cumulative вүҘ threshold
    в””в”Җв”Җ Bottom-most: add, mult, comparison, div, exp, FP arithmetic
